# Azure Architect Project

## Objective
This project is a hands-on Azure learning and portfolio project designed to build real-world cloud engineering skills while preparing for:
- AZ-104 Microsoft Azure Administrator
- AZ-305 Azure Solutions Architect Expert

## Current Progress
Completed so far:
- Resource group creation
- Virtual network creation
- Subnet planning
- Network security group basics
- Virtual machine setup
- SSH connectivity and troubleshooting

## Planned Build
This project will evolve into a production-grade Azure platform including:
- Segmented networking
- Secure administration access
- Storage services
- Identity and RBAC
- Monitoring and alerting
- Backup and disaster recovery
- Load balancing / application gateway
- Infrastructure as Code with Bicep

## Repository Structure
- `docs/` → architecture notes and exam mapping
- `infra/scripts/` → Azure CLI deployment scripts
- `infra/bicep/` → Infrastructure as Code
- `monitoring/` → alerts and dashboards
- `app/` → sample application layer

## Learning Goal
The goal is not only to deploy Azure resources, but to understand:
- what each service does,
- why it is used,
- when to choose it,
- and what trade-offs an architect should consider.
